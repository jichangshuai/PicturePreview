# PicturePreview
Description

picture preview like a gallery.

Step 1. Add the JitPack repository to your build file

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  Step 2. Add the dependency
  
  dependencies {
	        implementation 'com.github.jichangshuai:PicturePreview:0.1.2'
	}
